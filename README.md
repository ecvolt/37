# Ecovolt – Site Institucional

Site da Ecovolt, agência de marketing especializada em energia solar.

## 🚀 Como publicar no GitHub Pages

### 1. Crie um repositório no GitHub
- Acesse [github.com](https://github.com) e faça login
- Clique em **"New repository"**
- Dê um nome (ex: `ecovolt-site`)
- Deixe como **Public**
- Clique em **"Create repository"**

### 2. Suba os arquivos

Com Git instalado, rode no terminal:

```bash
git init
git add .
git commit -m "primeiro commit"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/ecovolt-site.git
git push -u origin main
```

> Substitua `SEU_USUARIO` pelo seu usuário do GitHub.

### 3. Ative o GitHub Pages
- No repositório, vá em **Settings → Pages**
- Em **Source**, selecione o branch **main** e a pasta **/ (root)**
- Clique em **Save**

### 4. Acesse o site
Após alguns minutos, o site estará disponível em:
```
https://SEU_USUARIO.github.io/ecovolt-site/
```
